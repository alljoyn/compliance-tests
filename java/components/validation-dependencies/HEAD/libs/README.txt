This project can be used to install the necessary AllJoyn dependencies into the local Maven repository. 

Please copy the following file(s) into the current libs folder before building the pom file.

about.jar
alljoyn.jar
alljoyn_app_utils.jar
alljoyn_config.jar
alljoyn_ns.jar
alljoyn_ns_commons.jar
alljoyn_ns_nativeplatform.jar
controlpanel.jar
controlpanel_adapter.jar
liballjoyn_java.so
onboarding.jar
