This project can be used to install the necessary AllJoyn dependencies into the local Maven repository. 

Alternatively, the validation-dependencies-dl project can be used to automatically download and extract the components from their release locations.

Please copy the following file(s) into the current libs folder before building the pom file.

alljoyn.jar
alljoyn-rel.jar (use release version of alljoyn.jar and rename it)
liballjoyn_java.so
liballjoyn_java-rel.so (use release version of liballjoyn_java.so and rename it)
alljoyn_about.jar
alljoyn_config.jar
alljoyn_apps_android_utils.jar
NotificationService.jar
NSCommons.jar
NSNativePlatformAndr.jar
alljoyn_onboarding.jar
ControlPanelService.jar
ControlPanelAdapter.jar
