CTT Web Server Version v1.5.0 Release Notes
===========================================
 
Fully Validated Platforms
-------------------------
 1) Windows 7 (64 bit); Internet Explorer version 11.0.9600.17728
 2) Windows 7 (64 bit); Google Chrome version 42.0.2311.90
 3) Windows 7 (64 bit); Firefox version 37.0.2

Features added in v1.5.0
------------------------
 1) CTT Web Server v1.5.0 works with CTT Local Agent v1.6.0, v1.7.0, v1.8.0 and v1.9.0.
 2) CTT Web Server v1.5.0 has a new web site organization and new logos according to new AllJoyn Certified program.
